Strategic Overview

Objective

Create an interactive internal learning platform for students to learn React JS and web technologies effectively through video tutorials, coding assignments, and progress tracking.

Goal

Enhance student engagement, provide structured learning pathways, and generate measurable outcomes that boost college reputation and admission rates.

Students (Primary), Faculty Members, and College Management seeking data-driven insights on learning effectiveness.

Improved learning outcomes, higher placement readiness, YouTube revenue generation, and enhanced college competitive positioning.

Problem Analysis

Without Portal

  • Fragmented learning resources scattered across platforms
  • No structured learning pathway or prerequisites
  • Manual tracking of student progress and assignments
  • Q&A sessions open to all, reducing effectiveness
  • Difficult to measure learning outcomes
  • Students lack motivation and engagement
  • No integration with industry tools (GitHub)

With Portal

  • Centralized platform with all learning resources
  • Step-by-step curriculum with enforced prerequisites
  • Automatic progress tracking via GitHub Classroom
  • Q&A eligibility based on topic completion
  • Comprehensive analytics dashboard for insights
  • Gamification with badges and leaderboards
  • Real-world GitHub workflow experience
